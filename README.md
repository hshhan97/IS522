# MSIS 522 HW1 — COVID-19 Mortality Prediction

This repository contains a complete end-to-end data science workflow for predicting COVID-19 patient mortality using tabular clinical data.

## Contents
- `MSIS_522_HW1_Submission.ipynb` — main notebook with EDA, predictive modeling, SHAP explainability, and artifact export
- `app.py` — Streamlit application with four required tabs
- `requirements.txt` — Python dependencies
- `saved_models/` — pre-trained model files generated by the notebook
- `artifacts/` — exported comparison table and figures used by the Streamlit app
- `covid.csv` — dataset file (place in repo root)

## Models
- Logistic Regression
- Decision Tree (GridSearchCV)
- Random Forest (GridSearchCV)
- LightGBM (GridSearchCV)
- Neural Network (Keras)

## How to run locally
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Open and run the notebook `MSIS_522_HW1_Submission.ipynb` to generate the trained models and artifacts.
3. Start the Streamlit app:
   ```bash
   streamlit run app.py
   ```

## Deployment note
Before deploying, make sure the following are committed to GitHub:
- `app.py`
- `requirements.txt`
- `README.md`
- `covid.csv`
- `saved_models/`
- `artifacts/`

After deployment, add your public Streamlit URL to your Canvas submission together with the GitHub repository link.
